# 43 信号检测器(SD)

信号检测器作为探测装置 与敏感电缆连接 用于处理信号并显示由干扰电压导致的故障 这种探测装置宜模拟工业制造的数字电气设备 比如 已用于船舶控制和自动化任务的存储可编程逻辑控制器（依据IEC60092-504）。

# 44 敏感 引流 电缆

敏感电缆用作信号检测器的接收器。电缆长度宜与试验场地的边长匹配。试验中宜使用以下两类电缆:

类型 非屏蔽双芯电缆 或  
类型 屏蔽四芯电缆 如 规定的屏蔽四芯电缆

# 45 干扰(源)电缆

干扰电缆的长度与敏感电缆相同 由接地快速瞬变脉冲群馈电 干扰电缆宜使用非屏蔽电缆 类型

# 46 脉冲群发生器

脉冲群发生器应根据 的规定产生快速瞬变脉冲群。

# A2 试验结果评估

# A21 概述

确定空间标准 以便在受电设备未发生故障的情况下在电缆最小敷设间距内实现最大脉冲群阈值在任何情况下 干扰电缆和敏感电缆的敷设要尽可能靠近地面

# A22 特殊情况

当单根电缆或一边为 类 另一边为 类的成束电缆平行敷设长度超过几米时 不同类别电缆之间保持最大可能敷设间距 且宜尽可能接近地面或距离地面高度不超过 $h = 5 0 ~ \mathrm { ~ m m }$ 当敷设间距$d = 5 0 \ \mathrm { m m }$ 、敷设高度 $h = 0 \ \mathrm { m m }$ 时，对于屏蔽四芯干扰电缆，最高阈值大于 $+ 4 \ 0 0 0 \ \mathrm { ~ V ~ }$ ，最低阈值小于$- 4 \ 0 0 0 \ \mathrm { V }$ 若为非屏蔽双芯电缆 阈值会有所减少 为使阈值大于 $2 \ 0 0 0 \ \mathrm { ~ V ~ }$ 敷设间距 $d$ 宜增加到$2 0 0 \ \mathrm { m m }$ 。

有必要通过进一步的测量,确认上述结果。
