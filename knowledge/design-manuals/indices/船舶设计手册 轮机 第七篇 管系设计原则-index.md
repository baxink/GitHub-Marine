## 管系分类速查

| 分类方式 | 类别 | 说明 |
|---------|------|------|
| **按功能** | 动力管路 | 燃油、滑油、冷却水、压缩空气、排气等 |
| | 船舶管路 | 舱底水、压载水、消防水、生活用水、液货等 |
| **按等级(I/II/III级)** | I级管系 | 高压/高温，如蒸汽>1.6MPa或>300℃ |
| | II级管系 | 中压/中温 |
| | III级管系 | 低压/低温 |

## 材料选择速查

| 材料 | 适用范围 | 备注 |
|------|---------|------|
| 无缝钢管 | I/II级管、蒸汽管、压缩空气管、燃油管 | CCS规范要求 |
| 焊接钢管 | 非I/II级管 | 需船级社认可工厂 |
| 无缝铜管 | 海水冷却、仪表管 | 铜镍管耐海水腐蚀 |
| 塑料管 | 疏排水、生活污水(非重要管路) | 温度≤60℃，IMO A.753(18) |
