# Knowledge Base

This directory stores reference materials, documentation, validated conclusions, and other citable knowledge resources.

## Structure

You can organize this however you prefer. Some suggested categories:

### `references/`
- Research papers and articles
- Technical documentation
- Standards and specifications

### `docs/`
- Process documentation
- How-to guides
- Organizational knowledge

## Usage

Organize files in whatever way makes sense for your work. The AI assistant will adapt to your organization style.

Validated conclusions from `research_研究/` should be migrated here with source traceability and a verification state when they become reusable.

**Flexible structure**: You can use single files or create folders for related materials. The workspace index tracks both folders and files in this directory.

## AI Integration

AI assistants can read and reference materials in this directory regardless of their format or structure.

For formal reviews, prefer this directory over `research_研究/` whenever authoritative citation is required.
