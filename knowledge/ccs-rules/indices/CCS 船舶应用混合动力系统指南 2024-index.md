# CCS 船舶应用混合动力系统指南 2024-index

> 摘要：CCS混合动力系统指导性文件（GD009-2024），涵盖并联/串联混合动力系统技术要求、产品检验及船舶检验要求。船长20m及以上混合动力船舶适用。

## 核心内容

| 章节 | 核心内容 |
|------|---------|
| **第1章 通则** | 适用范围（船长≥20m）、入级与检验依据、串联/并联混合动力系统定义 |
| **第2章 混合动力系统技术要求** | 并联混合动力特殊要求、串联混合动力特殊要求 |
| **第3章 混合动力装置产品检验** | 固态开关检验要求 |
| **第4章 混合动力船舶检验** | 建造中/建造后检验 |
