## 常用问法对应入口

- **问“2019原文怎么写”** → 直接进2019底座对应章节
- **问“2023改了哪里”** → 进2023关联索引
- **问“2025改了哪里”** → 进2025关联索引
- **问“现行要求是什么”** → 按本页表格，从2019开始顺读到2025
- **问“某主题现在是否仍适用2019原文”** → 先看本页该主题行，若2023/2025栏为空，通常仍以2019为主

## 说明

- 本总入口当前重点覆盖**第5篇 船舶安全**，因为现有2019主体切片已建立。
- 其他篇章虽已在2023/2025关联索引中标明入口，但若后续建立2019主体切片，建议继续按本总入口模式扩展。
